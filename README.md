# 👜 AstroBags – Explore Beyond Limits

AstroBags is a modern, visually rich single-page website designed to showcase innovative backpack collections including tech-enabled, eco-friendly, and lifestyle-based products.

---

## 🌌 Features

* 🌟 Animated **starry background effect**
* 🧭 Smooth navigation across product categories
* 🛍️ Multiple backpack categories:

  * Tech-Enhanced
  * Modular
  * Eco-Friendly
  * Fitness & Outdoor
  * Travel
  * Smart Backpacks
  * Ergonomic
  * Professional & Student
  * Fashion-Forward
  * Parent Backpacks
* 📦 Product cards with:

  * Images
  * Pricing
  * “More Info” modal
  * “Buy Now” confirmation modal
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (embedded styling)
* **Vanilla JavaScript**
* No external frameworks used

## ⚙️ How to Run

1. Download or clone the repo:

```bash
git clone https://github.com/your-username/astrobags.git
```

2. Open the folder:

```bash
cd astrobags
```

3. Run the project:

* Simply open `index.html` in your browser

---

## ✨ Key Highlights in Code

* ⭐ **Dynamic Star Generation**

  * JavaScript creates 500 animated stars for background effect

* 📦 **Product Modals**

  * Uses `<dialog>` element for:

    * Product details
    * Purchase confirmation

* 🎨 **Grid Layout**

  * CSS Grid used for category navigation
  * Flexbox used for product cards
